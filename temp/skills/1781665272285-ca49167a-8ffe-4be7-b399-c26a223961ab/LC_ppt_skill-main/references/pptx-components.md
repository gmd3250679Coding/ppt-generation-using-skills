# PPTX 可编辑组件库

组件参数按 16:9 宽屏设计, 坐标以 1280 x 720 画布为参考。使用 artifact-tool 或其他 PPTX API 时保持元素可编辑。

## 页面基础

- 安全边距: 左右 72, 顶部 48, 底部 56。
- 标题区: `x=72, y=48, w=920, h=54`。
- 内容区: `x=72, y=128, w=1136, h=500`。
- 页脚线: `x=72, y=662, w=1136, h=1`, 颜色 `#D6E6FF`。
- 页码: `x=1150, y=672, w=58, h=18`, 10 pt, `#667085`。

## 标题组件

- 标题: 30-34 pt, bold, `#111827`, 字体 `Microsoft YaHei`。
- 副标题或说明: 14-16 pt, `#667085`。
- 蓝色短横线: `x=72, y=106, w=54, h=5`, fill `#1D78FA`。
- 深色页标题使用白色 `#FFFFFF`, 短横线可用 `#A3C8F9`。

## 能力卡

- 卡片: fill `#F8FBFF`, border `#D6E6FF`, radius 6。
- 左侧强调条: `w=5`, fill `#1D78FA` 或 `#00A0A0`。
- 卡片标题: 18-20 pt, bold, `#111827`。
- 卡片正文: 13-15 pt, `#4D4D4D`, 2-4 行。
- 三列卡片: 单卡 `w=348, h=170`, gap 28。
- 四列卡片: 单卡 `w=260, h=160`, gap 24。

## 指标大字报

- 数字: 42-56 pt, bold, `#0D48CE`。
- 单位: 18-22 pt, `#1D78FA`。
- 说明: 13-15 pt, `#4D4D4D`。
- 指标卡背景用白色或 `#F8FBFF`, 避免大面积橙色。
- 没有真实数据时写 `待补充` 或 `示例`, 不编造口径。

## 流程节点

- 节点圆: 38-44 px, fill `#1D78FA`, 文本白色。
- 连接线: 2 px, `#A3C8F9`。
- 节点标题: 17-19 pt, bold, `#111827`。
- 节点说明: 12-14 pt, `#4D4D4D`, 不超过两行。
- 4 步流程优先横排; 5-6 步可分两行或压缩说明文字。

## 蓝底解释框

- 背景: fill `#0D48CE` 或 `#0E48CE`。
- 标题: 22-28 pt, bold, 白色。
- 正文: 14-16 pt, `#D6E6FF`。
- 可用一条青绿 `#00A0A0` 细线做次强调。

## 浅蓝说明条

- 背景: fill `#EEF6FF`。
- 左侧强调条: fill `#1D78FA`, `w=5`。
- 文本: 14-16 pt, `#111827`。
- 用于风险提示、建设建议、阶段目标, 不要连续多页重复。

## 表格

- 表头: fill `#0D48CE`, 白字, 12-14 pt。
- 单元格: 白底或极浅蓝交替, 10-12 pt。
- 边框: `#D6E6FF`。
- 列名保持短: 能力、现状、方案、价值、责任人、时间。
